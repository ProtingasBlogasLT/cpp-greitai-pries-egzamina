## Apie if dalį

Rašoma norint patikrinti ar palyginti vieną dalyką ir įvykdyti kodą, jei patikrinimas grąžins `true` tipo atsakymą.

## if sintaksė

```cpp
if (sąlyga)
{
    /* kodas, jei sąlyga yra teisinga (atsakymas į
    klausimą yra true) */
}
```
